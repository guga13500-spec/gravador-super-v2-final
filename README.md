# Gravador Super 2.0

O Gravador Super 2.0 é um aplicativo de gravação de voz profissional, desenvolvido sob medida para o Gustavo (Guga 8776).

## 🚀 Principais Mudanças na Versão 2.0

- **Identidade Exclusiva:** O aplicativo foi renomeado de "Fossify Voice Recorder" para **Gravador Super**.
- **Pacote Customizado:** Alterado para `com.guga8776.gravadorsuper`, garantindo uma instalação única e sem conflitos.
- **Interface Limpa:** Removidas todas as menções a marcas de terceiros nas telas principais.
- **Tradução Aprimorada:** Interface e documentação interna totalmente em Português do Brasil.
- **Edição de Áudio:** Recurso de corte (trim) integrado para refinar suas gravações.
- **Controle de Velocidade:** Reproduza seus áudios em 0.5x, 1.0x, 1.5x ou 2.0x.
- **Acessibilidade:** Suporte completo ao TalkBack, otimizado para navegação por voz.
- **Segurança:** Sistema de Lixeira para evitar perdas acidentais.

## 🛠 Como Compilar

Este é um projeto Android padrão usando Gradle (Kotlin DSL).

1. Abra o projeto no **Android Studio**.
2. Sincronize o Gradle.
3. Gere o APK através do menu `Build > Build Bundle(s) / APK(s) > Build APK(s)`.
